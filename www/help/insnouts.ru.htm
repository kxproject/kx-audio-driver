Входы и выходы

Вкладка kX микшера входы и выходы позволяет управлять следующими входами и выходами (обратите внимание, что в зависимости от модели вашей звуковой платы, некоторые входы могут быть не отображены).

Выходы:

Аналоговой фронт
Этот фэйдер управляет уровнем передних аналоговых выходов (стерео).

Аналоговый тыл
Этот фэйдер управляет уровнем тыловых аналоговых выходов (стерео).

Аналоговый центр
Этот фэйдер управляет уровнем центрального аналогового выхода (только платы 5.1 и Audigy).

Аналоговый сабвуфер
Этот фэйдер управляет уровнем выхода сабвуфера (только платы 5.1 и Audigy).

SPDIF
Этот фэйдер управляет уровнем SPDIF выхода (цифровой стерео)

Наушники
Этот фэйдер управляет уровнем выхода наушников (аналоговый стерео - только LiveDrive).

A1/A2 выход
Этот фэйдер управляет уровнем Analog-1 и Analog-2 выходов (аналоговый стерео - только APS)

A3/A4 выход
Этот фэйдер управляет уровнем Analog-3 и Analog-4 выходов (аналоговая стерео пара - только для плат APS с MOC)

A5/A6 выход
Этот фэйдер управляет уровнем Analog-5 и Analog-6 выходов (аналоговая стерео пара - только для плат APS с MOC)

A7/A8 выход
Этот фэйдер управляет уровнем Analog-7 и Analog-8 выходов (аналоговая стерео пара - только для плат APS с MOC)

Входы:

AC97 вход
Этот фэйдер управляет уровнем сигнала, поступающего из AC97 кодека (аналоговые моно и стерео входы).

Цифровой CD вход
Этот фэйдер управляет уровнем цифрового CD входа (цифровой стерео SPDIF).

I2S вход
Этот фэйдер управляет уровнем I2S входа (цифровой стерео I2S - только LiveDrive).

Оптический вход
Этот фэйдер управляет уровнем оптического входа (цифровой стерео SPDIF - только LiveDrive).

Mic2 вход
Этот фэйдер управляет уровнем Mic2 входа (аналоговый моно - только LiveDrive).

Coaxial вход
Этот фэйдер управляет уровнем коаксиального входа (цифровой стерео - только LiveDrive).

Aux2 вход
Этот фэйдер управляет уровнем Auxiliary-2 входа (аналоговый стерео - только LiveDrive).

A1/A2 вход
Этот фэйдер управляет уровнем Analog-1 и Analog-2 входов (аналоговая стерео пара - только APS).

A3/A4 вход
Этот фэйдер управляет уровнем Analog-3 и Analog-4 входов (аналоговая стерео пара - только APS с Edrive).

D1 вход
Этот фэйдер управляет уровнем Digital-1 входа (цифровой стерео - только APS)

D2 вход
Этот фэйдер управляет уровнем Digital-2 входа (цифровой стерео - только APS с Edrive)

Дополнительную информацию можно найти в нашем руководстве по назначению входов и выходов (I/O Assignment Guide).

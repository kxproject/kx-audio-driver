Контроль уровня записи

Вкладка записи окна kX Микшера обеспечивает доступ к следующим регулировкам уровня записи:

Программные источники:

Общий уровень записи
Эта стерео пара управляет общим уровнем записи для всех источников.

Уровень записи Wave
Этот моно фэйдер управляет уровнем записи PCM Wave источников.

Уровень записи синтезатора
Этот моно фэйдер управляет уровнем записи MIDI Синтезатора.

Уровень записи FX1 (Ревер)
Этот моно фэйдер управляет количеством FX1 (Ревера по умолчанию) добавляемого при записи к воспроизводимым источникам.

Уровень записи FX2 (Хорус)
Этот моно фэйдер управляет количеством FX2 (Хоруса по умолчанию) добавляемого при записи к воспроизводимым источникам.

Аппаратные источники:

Уровень записи AC97
Эта стерео пара управляет уровнем записи AC97 источника(ов)

Уровень записи CD Digital
Эта стерео пара управляет уровнем записи цифрового входа CD Digital (CD_SPDIF).

Уровень записи I2S
Эта стерео пара управляет уровнем записи цифрового входа I2S (только для систем с LiveDrive).

Уровень записи Optical
Эта стерео пара управляет уровнем записи оптического цыфрового входа (только для систем с LiveDrive).

Уровень записи Mic-2 / Line-2
Эта стерео пара управляет уровнем записи источников Mic-2 / Line-2 (только для систем с LiveDrive).

Уровень записи Coaxial
Эта стерео пара управляет уровнем записи коаксиального цифрового входоа (только для систем с LiveDrive).

Уровень записи Aux2
Эта стерео пара управляет уровнем записи второго дополнительного входа Aux2 (только для систем с LiveDrive).

Уровень pаписи Analog-1/Analog-2 (только для плат APS)
Эта стерео пара управляет уровнем записи аналоговых источников A1/A2.

Уровень записи Analog-3/Analog-4 (только для плат APS с Edrive)
Эта стерео пара управляет уровнем записи аналоговых источников A3/A4.

Уровень записи цифрового входа D1 (только для плат APS)
Эта стерео пара управляет уровнем записи цифрового входа D1.

Уровень записи цифрового входа D2 (только для плат APS с Edrive)
Эта стерео пара управляет уровнем записи цифрового входа D2.

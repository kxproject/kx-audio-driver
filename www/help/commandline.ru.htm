kX Справочник консольных команд

kxsetup.exe опции Запускает стандартный kX драйвер и настройки kX микшера 
--clean
 Выполняет процедуру очистки перед инсталяцией, которая выполнится автоматически при установленных kX драйверах. Отключает некоторые программные компоненты Creative и любые предыдущие установленные компоненты kX микшера и kX драйвера. 
--noreboot
 Отменяет перезагрузку после инсталяции драйвера.
kxmixer.exe опции: 
--quit
 Закрывает все активные компоненты kX микшера 
--once
 Автоматически выгрузить микшер после восстановления настроек (необходимо использовать совместно с '--startup') 
--startup
Исполняет последовательность ввода в действие: восстановливает kX параметры настройки из файла kxdefault.kx 
--shell
Опция интеграции оболочки: только для внутреннего использования 
--no10k2
Отключает поддержку 10k2 
--10k2
Принудительная поддержка 10k2 
--no51
Отключает поддержку 5.1 
--51
Принудительная поддержка 5.1 
--aps
Принудительная поддержка E-mu APS 
--noaps
Отключает поддержку E-mu APS 
--noa2
Отключает поддержку Audigy2 
--a2
Принудительная поддержка Audigy2 
--nok2ex
Отключает поддержку Audigy2 Platinum Ex 
--k2ex
Принудительная поддержка Audigy2 Platinum Ex 
--no2k
Отключает поддержку Windows 2000/XP Extended GUI
